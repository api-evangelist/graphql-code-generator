# GraphQL Code Generator

GraphQL Code Generator is a plugin-based code generation tool from The Guild that generates TypeScript types, React hooks, Angular services, resolvers, and SDKs directly from GraphQL schemas and operation documents. It supports 50+ plugins and targets frontend frameworks (React, Vue, Angular, Svelte) as well as server-side resolvers (Apollo Server, GraphQL Yoga) and custom output formats.

## About

GraphQL Code Generator was created by Dotan Simha and is maintained by The Guild, the largest open-source group in the GraphQL ecosystem. The tool is MIT-licensed and freely available on npm under the `@graphql-codegen` scope.

Key capabilities:
- End-to-end TypeScript type safety from schema to client
- Typed queries, mutations, and subscriptions for all major frontend frameworks
- Typed server-side resolver generation for Node.js and Java
- Fully typed SDK generation
- Extensible plugin architecture — write custom plugins to generate any output
